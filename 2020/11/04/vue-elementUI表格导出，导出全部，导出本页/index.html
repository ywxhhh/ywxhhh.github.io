  1. 导出本页

    • 页面引入这个函数
      import { downLoad, downLoadAll } from '@/utils/exportExcel'

      这里的坑真的是触不及防，先来说说问题，之前使用都正常，然后突然新增了一个需求，然后发现真是快乐，bug来了。本来导出的表格，都没有设置fixed属性，所以正常。然后这次导出的表格有这个属性然后bug就这么触不及防的出现了—导出的表格有重复数据，然后怎么办，改呗，主要是判断要导出的节点中是否有fixed的表格，如果有，转换excel时先将该dom移除，然后append回去，详情见代码咯，愉快解决~溜了溜了

  2. 导出全部
