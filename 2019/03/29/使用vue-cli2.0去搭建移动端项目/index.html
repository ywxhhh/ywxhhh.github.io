使用vue-cli2.0去搭建移动端项目

三月 29, 2019 4555

使用脚手架去搭建一个vue项目，其中包括webpack打包时要修改的内容，还有用到的框架，和使用vm的移动端适配，还有一厘米像素差

1. 安装vue-cli

  1. 使用npm（需要安装node环境）全局安装webpack，打开命令行工具输入：npm install webpack -g或者（npm install -g webpack），安装完成之后输入 webpack -v，如果出现相应的版本号，则说明安装成功。
  2. 全局安装vue-cli，在cmd中输入命令 : npm install --global vue-cli,安装完成之后输入 vue -V（注意这里是大写的“V”），如果出现相应的版本号，则说明安装成功。

2. 用vue-cli来构建项目

  1. 在你要建的盘，新建一个文件夹， cd (文件夹名称)
  2. 配置完成后，可以看到目录下多出了一个项目文件夹baoge，然后cd进入这个文件夹
